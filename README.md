# Vehicle Data Cleaning and Visualization Project

## Objective
The objective of this project is to clean a raw vehicle dataset and visualize useful insights using Python.

## Tools Used
- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Vehicle Dataset containing vehicle information such as:
- Brand
- Model
- Price
- Mileage
- Fuel Type
- Year
- Transmission

## Data Cleaning Performed
- Handled missing values
- Removed duplicate records
- Checked data types
- Prepared data for analysis

## Visualizations
1. Top Vehicle Brands
2. Fuel Type Distribution
3. Vehicle Price Distribution
4. Mileage vs Price
5. Vehicles by Year
6. Correlation Heatmap

## Key Findings
- Some brands dominate the dataset.
- Higher mileage vehicles generally have lower prices.
- Gasoline vehicles are the most common.
- Most vehicles belong to recent manufacturing years.

## Outcome
Learned data preprocessing, visualization and data storytelling techniques.
